Login

Al abrir el programa nos vemos forzados a someternos a una prueba de verificación con una cuenta de usuario y una contraseña.

Si la persona que desea ingresar al sistema no cuenta con uno de estos requisitos, se le será imposible acceder a sistema de inventario Fly.

Una vez ingresado el cargo, nombre de usuario y la contraseña, el sistema verificará si existe esa cuenta con las características dadas por el operador y determinará si podrá acceder o no.


Inicio

Una vez ingresado al sistema, podrá gozar de todos las opciones que su cargo le permite, ya que solo el administrador tendrá un acceso total al sistema Fly.

En la esquina superior derecha, se podrá visualizar el usuario que ha ingresado y su cargo. También se mostrará una opción la cual premite al usuario cerrar sesión sin necesidad de cerrar el programa. Una vez cerrada la sesión, se deberá volver a ingresar usuario y contraseña con su respectivo cargo.

En la esquina superior izquierda, se encuentra un botón para poder minimizar la aplicación cuando el usuario lo requiera.


En la esquina inferior derecha se muestra un menú de opciones, las cuales cuenta con todas las diferentes partes del sistema. También contiene un botón de salir que cerrará la aplicación de manera inmediata, una opción de sobre mí que hace referencia al manual de usuario que usted está leyendo en estos momentos.

Formulario de Proveedores

Te permite guardar los datos que ingreses, modificar un proveedor, o eliminarlos. Para poder ingresar un nuevo proveedor se deberá hacer lo siguiente:
1. Como primer paso se deberá presionar el botón de Nuevo.
2. Luego, se tendrán que llenar todos los campos para que se guarden.
3. Por último, se presionará el botón Agregar.
Al final mostrará un mensaje de Proveedor Agregado.

Formulario de Productos

Te permite guardar los datos que ingreses, modificar un producto, o eliminarlos. Para poder ingresar un nuevo producto se deberá hacer lo siguiente:
1. Como primer paso se deberá presionar el botón de Nuevo.
2. Luego, se tendrán que llenar todos los campos para que se guarden.
3. Por último, se presionará el botón Agregar.
Al final mostrará un mensaje de Producto Agregado.
*Se recomienda escribir el código al final, ya que se podrá visualizar la descripción completa del producto.

Formulario de Compras



Para poder ingresar una nueva compra, se deberá hacer lo siguiente:
1. Como primer paso se deberá ingresar el código de producto.

*En caso de no conocerlo, se puede ir a la pestaña "Buscar Producto" (imagen dos) y escribir la descripción del producto para poder buscar su código.

2. Luego, ingresar la cantidad a comprar.
3. Después de ingresar la cantidad, se deberá presionar el botón "+" para agregar la compra.
4. Como último paso, al final de ingresar todas las compras realizadas, se deberá presionar el botón "Guardar" para poder tener registro de la compra realizada.
Al final mostrará un mensaje de Compra Realizada.

*Se mantendrá registro de el usuario que realizó la compra, fecha y el total que se invirtió, así como también los productos y las cantidades que se compraron.

Formulario de Ventas





Para poder ingresar una nueva venta, se deberá hacer lo siguiente:
1. Como primer paso se deberá ingresar el RTN del cliente.

*En caso de no estar registrado en el sistema, aparecerá un mensaje de si desea registrar cliente, luego deberá llenar los datos (imagen dos); caso contrario, se llenarán las casillas automáticamente.

2. Luego, una vez lleno los datos del cliente, deberá ingresar el código del producto.

*En caso de no conocerlo, se puede ir al botón "Buscar" (imagen tres) y escribir la descripción del producto para poder buscar su código.

3. Después de ingresar el código de producto, se deberá presionar únicamente la tecla "Enter" para poder llenar las otras casillas.

*Queda a opción del usuario si quiere o no modificar el precio del producto.

4. Una vez definido el precio del producto, se deberá ingresar una cantidad (la cantidad que aparece por defecto, es la existencia actual de ese producto)

*En caso de que se ingrese una cantidad mayor a la cantidad que hay en existencia, aparecerá un mensaje diciendo que no hay suficiente producto.

*Se mantendrá registro de el usuario que realizó la venta, fecha y el total que se cobró, así como también los productos y las cantidades que se vendieron.
